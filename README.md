# ShameBell / Срачомер

Колокол позора для ваших групповых чатов.

Этот бот-стукач следит за беседой и с помощью AI определяет, когда участники начинают обсуждать «запретные» темы.

![0 days without... meme](https://images.meme-arsenal.com/4de2dbb28e7c1ecde0492a6eb4ecaffa.jpg)

# Как это работает:

**1. Слежка**: Бот читает все сообщения и отправляет их на проверку нейросети.

**2. Срам**: Если запретная тема обнаружена, бот сбрасывает счетчик и пишет в чат: «Дней без обсуждения <плохая_тема>: 0».

**3. Похвала**: Если все ведут себя прилично, счетчик дней увеличивается. Участники могут узнать свой рекорд по команде и гордиться собой.

# Зачем это нужно?

Чтобы превратить срач в игру и снизить токсичность через публичный позор в шутливой форме.

# Это пет проект

В этом проекте я применяю Clean Architecture и DI/IoC. Это очевидный оверинженеринг для tg-бота такого масштаба. **Это сделано намеренно** просто потому что это пет и я делаю его для удовольствия и освоения интересностей. А еще мне просто нравится "клин аркитекча".

# Запуск

```sh
git clone https://github.com/osipovts/shamebell.git
cd shamebell
cp .env.example
nano .env # задайте токен вашего тг-бота
docker compose up app-prod --build
```
