# ShameBell / Срачомер

Колокол позора для ваших групповых чатов.

Этот бот-стукач анализирует сообщения в вашем групповом чате и определяет, когда участники начинают обсуждать «запретные» темы.

![0 days without... meme](https://images.meme-arsenal.com/4de2dbb28e7c1ecde0492a6eb4ecaffa.jpg)

# Как это работает:

**1. Слежка**: Бот читает все сообщения и анализирует их.

**2. Срам**: Если запретная тема обнаружена, бот сбрасывает счетчик и публично позорит негодяя, устроившего заварушку.

**3. Похвала**: Если все ведут себя прилично, счетчик дней увеличивается. Участники чата могут узнать свой групповой рекорд и гордиться собой.

# Зачем это нужно?

Чтобы превратить срач в игру и снизить токсичность через "колокол позора" в шутливой форме.

# Запуск

```sh
git clone https://github.com/osipovts/shamebell.git
cd shamebell
cp .env.example
nano .env # задайте токен вашего тг-бота
docker compose up app-prod --build
```
